# GedcomX Angular App
## Functionality of the App
- View entity lists
- Edit entities
- add new entities
- show relations to others in a chart
## Navigation
- Start page
- Persons List
- Places List
- Relationships List
- Further root arrays
## Start Page Content
- Description of the App
- Loading GedcomX JSON
- Links to sub sites
## Person List Functionality
- List all Persons
- Query for Persons
 - By Name
 - By Date
- Add new Person
- Open person to edit
- Open Child Relations
- Open Parent Relations
## Places List Functionality
- List all Places
- Query for Place
  - by name
  - by container place
- Add new Place
- Open place to edit
